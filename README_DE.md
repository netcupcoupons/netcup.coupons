## Anleitung zum Einlösen
### Option A: Für produktspezifische Gutscheine (RS, VPS, Hosting)
1. **Warenkorb aufrufen:** Navigieren Sie direkt zur Netcup Warenkorbseite unter [netcup.com/de/checkout/warenkorb](https://www.netcup.com/de/checkout/warenkorb).
2. **Code eingeben:** Suchen Sie unten das Feld "Gutschein einlösen" und fügen Sie Ihren kopierten Gutscheincode ein.
3. **Rabatt anwenden:** Klicken Sie auf Einlösen. Der Rabatt wird sofort berechnet und vom Gesamtbetrag abgezogen.
4. **Bestellung abschließen:** Überprüfen Sie den Preis und klicken Sie auf Weiter, um den Kauf abzuschließen.

### Option B: Für den allgemeinen 5,00 € Rabattgutschein
1. **Tarif auswählen:** Legen Sie den gewünschten Webhosting-, VPS- oder Root-Server-Tarif in den Warenkorb.
2. **Gutscheinfeld suchen:** Suchen Sie während des Bestellvorgangs auf der Produktübersichtsseite nach dem Eingabefeld.
3. **Code anwenden:** Fügen Sie den generierten 5 € Code ein und bestätigen Sie ihn. Die 5,00 € werden abgezogen.
4. **Bestellung abschicken:** Überprüfen Sie Ihre Daten und schließen die Bestellung ab. Gutscheine sind nicht kombinierbar.

## Häufig gestellte Fragen (FAQs)
#### Q1: Wie löse ich einen Netcup-Gutschein ein?
A1: Klicken Sie auf den gewünschten Gutscheincode, um ihn zu kopieren. Fügen Sie ihn dann beim Bezahlvorgang im Warenkorb ein.
#### Q2: Sind die Gutscheine kostenlos?
A2: Ja. Die Generierung und Nutzung der Rabattcodes ist für Sie vollkommen gratis.
#### Q3: Werden die Gutscheincodes regelmäßig überprüft?
A3: Jeder Code wird automatisch durch unsere Skripte geprüft. Wir gleichen den Status alle 5 Minuten mit Netcup ab.
#### Q4: Können Gutscheine miteinander kombiniert werden?
A4: Nein. Pro Bestellung kann bei Netcup immer nur ein einziger Gutscheincode eingelöst werden.
#### Q5: Gelten die Rabatte auch für bestehende Kunden?
A5: Ja. Unsere Gutscheincodes für Webhosting, VPS und Root-Server können sowohl von Neukunden als auch von Bestandskunden genutzt werden. Einzige Ausnahme ist der separate 6 € Neukunden-Sofortgutschein.
